*******************************************************************************
**                                                                           **
**   Obtaining the documentation of the whole source code of the             **
**   kernel module for ADS7924                                               **
**                                                                           **
*******************************************************************************

1) Remain or navigate in this directory where this file "readme.txt" resides.

2) Type doxygen.

3) Open the file ./doc/html/index.html with your preferred web-browser.


Option: Integrating this documentation in to the eclipse help-system
===============================================================================

4) Follow the above mentioned steps 1 - 2.

5) Make a symbolic link in the hidden directory of eclipse:

   <your home directory>/.eclipse/org.eclipse.platform<version-number>/plugins/html
   -->
   <source directory of talker>/doc/html

6) Restart eclipse

You will find the documentation of the talker in the eclipse menu:
   help -> "Help Contents"

NOTE: Eclipse (version: 3.8.1) is a bit faulty in showing this documentation,
      you'd better to use a web-browser described above.
